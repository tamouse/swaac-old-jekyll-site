A one-line password generator

This is a simple, one line shell statement to generate a random password:

cat /dev/urandom | strings | grep -o '[[:alnum:]!@%_-+=., ]' | head -n 15 | tr -d '\n'

which gives something like:

oJhh!j0bNAiZNDa
